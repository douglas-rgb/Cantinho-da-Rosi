# 🌐 Projeto Web - Portfólio / Site

Este projeto é um site moderno e responsivo desenvolvido utilizando
**React + Vite**. O objetivo do projeto é apresentar serviços,
informações da empresa, contato e localização de forma profissional.

------------------------------------------------------------------------

# 🚀 Tecnologias Utilizadas

-   React.js
-   Vite
-   JavaScript
-   HTML5
-   CSS Modules

# 📦 Instalação

Clone o repositório:

    git clone https://github.com/seu-usuario/seu-projeto.git

Entre na pasta do projeto:

    cd seu-projeto

Instale as dependências:

    npm install

------------------------------------------------------------------------

# ▶️ Executando o Projeto

Para iniciar o servidor de desenvolvimento:

    npm run dev

Abra no navegador:

    http://localhost:5173
